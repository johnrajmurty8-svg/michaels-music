# Michael's Music

A heritage collection of handwritten guitar sheet music — digitised as an interactive web experience.

## Features
- 66 songs across 7 albums with chord/lyric sheets
- Vinyl record crate navigation with sleeve animations
- Live Session tab — curate up to 8 songs, drag-and-drop ordering, fullscreen presentation mode
- Turntable with tonearm animation and audio playback
- Original manuscript image viewer
- Fully responsive (desktop, tablet, phone)

## Deployment
This is a static single-page app. Deploy to Vercel by connecting this repo.
